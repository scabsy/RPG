new (startAngle, endAngle, radius, strokeWidth, closed, pie)
Constructor function of Arc module

Parameters:

  • startAngle int the starting angle of the arc
  • endAngle int the ending angle of the arc
  • radius int radius of the circle arc
  • strokeWidth int the thicknes of the arc in pixels
  • closed bool if true, draws a closing line between beginning and end of the arc
  • pie bool if closed and pie is true then draw a wedge
