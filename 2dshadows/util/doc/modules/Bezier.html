CalculatePoint (t, p0, p1, p2, p3)
Caclulate a point on a cubic Bézier curve

Parameters:

  • t number 0.0 is the starting point and 1.0 would be the end point
  • p0 number First point of Curve, a table with fields .x and .y, a Vector2D table is also possible
  • p1 number Second point of Curve, a table with fields .x and .y, a Vector2D table is also possible
  • p2 number Third point of Curve, a table with fields .x and .y, a Vector2D table is also possible
  • p3 number Fourth point of Curve, a table with fields .x and .y, a Vector2D table is also possible

Returns:

    a Vector2D table with the caclulated points
