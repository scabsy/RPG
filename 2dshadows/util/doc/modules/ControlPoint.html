new (optParent[, optX=0][, optY=0][, optHandleX=0][, optHandleY=0], optColor)
Create and draws a control point for a Bézier curve

A control point consists of the point itself (its position) an a curve handle The Beziér curve goes through the control point. The curve can go through the curve handle, but it must not. The curve handle controls the curviness.

Parameters:

  • optParent the parent object
  • optX number x position of the control point (default 0)
  • optY number y position of the control point (default 0)
  • optHandleX number x position of the curve handle (default 0)
  • optHandleY number y position of the curve handle (default 0)
  • optColor the color of the control point visualization in r,g,b,a values (default {0,0,1,1} = blue color)

Returns:

    GroupObject
