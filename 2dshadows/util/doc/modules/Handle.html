new (optParent[, optX=0[, optY=0]])
Create and draws a curve handle for a control point of a Bézier curve The curve handle controls the curviness.

Parameters:

  • optParent the parent object (in general a ControlPoint)
  • optX number x position of the curve handle (default 0)
  • optY number y position of the curve handle (default 0)

Returns:

    GroupObject
